# 葉來葉棒 AI 伴學自主學習 OS

GitHub repository name: `115509`

五專第115509組 葉來葉棒系統手冊 的專業化 GitHub Pages 靜態展示版。

## 展示定位

- 中文化、免登入、全功能展示。
- 根據學生文件的原始構想延伸成更接近正式產品的操作介面。
- 純前端靜態檔案，不連接後端、資料庫、第三方 API 或真實校內服務。
- 使用模擬資料，避免公開學生個資、帳密、內部網址與敏感資訊。

## 檔案

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## GitHub Pages

推送至 GitHub 後可於 Settings > Pages 設定：

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
